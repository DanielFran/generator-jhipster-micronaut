# Greetings, Micronaut Hipster!

This project generates a Micronaut based JHipster application üòé.

* Monolith projects
* JWT or OAuth 2.0 Authentication
* SQL Database Support

  * PostgreSQL
  * H2
* Ehcache
* Maven or Gradle Build System
* Angular or React Client
* Protractor Tests
